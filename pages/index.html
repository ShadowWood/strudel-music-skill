About

strudel-music is a Claude Code skill that bridges natural-language descriptions and live, generative music. Describe the mood, tempo, and character you want — “a warm lo‑fi beat with a dusty vinyl feel” — and the skill translates your words into executable Strudel code, ready to play in your browser.

Built on the official strudel.cc REPL, every pattern is streamed into the player via a URL hash. No build step, no npm packages — just describe, generate, and play.

Features

Natural‑language music

Describe what you hear in plain English — the skill generates the matching Strudel pattern.

Strudel REPL integration

Code runs inside the official strudel.cc REPL via an iframe — no local synthesis engine needed.

Live preview in browser

Vite HMR reloads the iframe on every save — hear changes in under a second.

Multi‑genre support

Built‑in templates for lo‑fi, hip‑hop, techno, ambient, drum & bass, and more.

Supported Genres

Each genre includes curated rhythm and tonal patterns — mix, layer, and tweak to create something uniquely yours.

Lo‑fi Hip‑hop Techno Ambient Drum & Bass House Jazz Funk

Quick Start

Install the skill, describe what you hear, and listen — three steps.

# 1. Install the skill into your agent bash install.sh

# 2. Start the Vite dev server bash scripts/init.sh

# 3. Open http://localhost:5173/ in your browser # and play the default pattern

# Once running, tell your agent: "make a chill lo‑fi beat with a warm vinyl feel"

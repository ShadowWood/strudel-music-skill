strudel-music / @strudel/web CDN diagnostic

This file is a self-contained scratchpad for any future move away from the strudel.cc iframe approach. It loads @strudel/web (and friends) from a CDN and exposes seven labeled test cases. Click "init" below once (browser autoplay policy), then run each test in turn. The expected verdict for each case is in the card description.

Test 2evaluate simple

Expected: evalScope accepts a one-line expression and schedules audio. No $: wrapper, no setcpm, no stack. If this works but the marker-syntax case does not, the issue is with declaration-marker handling.

(no output yet)

Test 3evaluate `$:`

Expected: evalScope handles the $: declaration marker the same way the strudel.cc REPL does. This is the same syntax the skill's pattern.js templates use, so it MUST work for any @strudel/web-based replacement of the iframe. 

$: s("hh*8")
(no output yet)

Test 4evaluate setcpm

Expected: setcpm is in scope and changes the tempo of the following pattern. If this throws "setcpm is not defined", the evalScope is missing the global helper and is NOT a drop-in replacement for the strudel.cc REPL. 

setcpm(60)
$: s("bd sd")
(no output yet)

Test 5sound without samples

Expected: a built-in oscillator-based sound plays, with NO bank downloads required. If this works but a sample-based test does not, the failure is in the prebake / sample-bank path, not in the core WebAudio pipeline. 

$: s("triangle").note("c3 e3 g3 c4")
(no output yet)

Test 6synth note

Expected: a short melodic phrase on the default synth. If the synth engine is broken, every higher-level pattern in the skill is also broken, so this is a high-priority test. 

note("c4 e4 g4 c5").s("square").lpf(800)
(no output yet)

Test 7stack

Expected: three layers (drums, bass, lead) play in parallel without errors. If stack is missing, this is the test that will fail first; this is also the most common shape in examples.md and genres.md. 

stack(
  s("bd*2, sd*2, hh*8"),
  note("c2 c2 g1 g1").s("sawtooth"),
  note("c4 e4 g4 e4").s("square")
)
